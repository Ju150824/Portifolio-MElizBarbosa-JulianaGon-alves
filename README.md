📌 Descrição
O projeto consiste na criação de um site de portfólio pessoal contendo informações sobre as integrantes, suas habilidades, cursos e um formulário de contato.

 🧱 Estrutura do Projeto
- `index.html` → Página inicial
- `sobre.html` → Informações sobre as integrantes
- `habilidades.html` → Habilidades técnicas e pessoais
- `cursos.html` → Cursos e formações
- `contato.html` → Formulário de contato
- `style.css` → Estilo do site
- `script.js` → Funcionalidades em JavaScript

⚙️ Funcionalidades
- Menu hambúrguer interativo
- Validação de formulário
- Contador de caracteres no campo de mensagem
- Layout responsivo para celular
- Navegação entre páginas

🎨 Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

📱 Responsividade
O site foi desenvolvido para funcionar tanto em computadores quanto em dispositivos móveis.

 🚀 Como executar
Basta abrir o arquivo `index.html` em um navegador.


Projeto acadêmico desenvolvido para fins educacionais.
